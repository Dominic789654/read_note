# AI/LLM 文章阅读记录

## 📚 目录说明

- `articles/` - 文章笔记存放目录
- `templates/` - 笔记模板
- `CATEGORIES.md` - 分类体系（动态更新）
- `index.md` - 总索引

## 🏷️ 分类体系

所有分类代码见 [CATEGORIES.md](./CATEGORIES.md)

## 📝 文件命名格式

```
[分类码]_序号_标题.md
例如：[LLM]_001_Transformer架构详解.md
```

## 📋 笔记模板

新增文章时使用 `templates/article_note_template.md`

## 🔍 快速查找

查看 [index.md](./index.md) 获取所有文章列表

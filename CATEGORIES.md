# 分类体系

> 最后更新：2025-02-01

## 分类代码规则

- 格式：大写字母缩写
- 可动态扩展
- 支持多级分类

## 当前分类

### 一级分类

| 代码 | 名称 | 说明 |
|------|------|------|
| LLM | 大语言模型 | LLM相关技术、模型、应用 |
| ARCH | 模型架构 | Transformer、注意力机制等架构设计 |
| TRAIN | 训练优化 | 预训练、微调、对齐方法 |
| AGENT | AI Agent | 智能体、工具调用、规划等 |
| RAG | 检索增强 | RAG技术、向量数据库等 |
| SAFETY | 安全对齐 | 安全性、红队测试、对齐技术 |
| APP | 应用实践 | 实际应用案例、产品落地 |
| PAPER | 论文研读 | 重要论文解读 |
| BLOG | 技术博客 | 技术文章、博客总结 |
| NEWS | 行业动态 | 新闻、报告、趋势分析 |

### 二级分类（按需扩展）

#### LLM - 大语言模型
- `LLM.BASE` - 基础模型（GPT、LLaMA等）
- `LLM.CAP` - 能力研究（推理、代码、多模态等）
- `LLM.EVAL` - 评估方法

#### ARCH - 模型架构
- `ARCH.ATTN` - 注意力机制
- `ARCH.TRANS` - Transformer变体
- `ARCH.MOE` - 混合专家模型

#### TRAIN - 训练优化
- `TRAIN.PT` - 预训练
- `TRAIN.SFT` - 有监督微调
- `TRAIN.RL` - 强化学习对齐（RLHF/RLAIF）

...（持续扩展中）
